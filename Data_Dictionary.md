# Data Dictionary

| Column Name | Description |
|------------|-------------|
| Customer_ID | Unique customer ID |
| Name | Customer name |
| Gender | Customer gender |
| Date_of_Birth | Customer DOB |
| City | Customer city |
| Join_Date | Customer joining date |
| Purchase_Amount | Purchase value |
| Email | Customer email |
| Customer_Age | Age derived from DOB |
| Purchase_Category | Low/Medium/High spender |
